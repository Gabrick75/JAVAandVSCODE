# Project Setup / Configuração do Projeto

## English
### About

This project provides a simple setup script that installs Java JDK, Visual Studio Code, essential Java extensions, Maven, Gradle, and Git on Linux (Ubuntu/Debian).  
It also creates a sample project folder `PROJETOS/TEST` with a Java file `TestSetup.java` for testing in VS Code.

### Installation

To set up the Java development environment on Linux (Ubuntu/Debian), follow these steps:

1. **Clone the project repository:**

```bash
git clone https://github.com/Gabrick75/JAVAandVSCODE
```
2. Go to the project folder:
```bash
cd JAVAandVSCODE
```
3. Make the setup script executable:
```bash
chmod +x script.sh
```
4. Run the setup script:
```bash
./script.sh
```
This will:

- Check disk space

- Update the system

- Install Java JDK

- Install VS Code and essential Java extensions

- Create PROJETOS/TEST/TestSetup.java on your Desktop
